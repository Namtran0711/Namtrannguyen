# Tran Nguyen Nam — Portfolio

A modern, responsive portfolio website built with React, TypeScript, and Tailwind CSS.

## Features

- Experience, Key Projects, Skills, and Reviews tabs
- Project type badges (Product / Internal Product / Campaign)
- Status indicators (Live / Coming Soon / Closed)
- Collapsible links & media section per project
- Inline YouTube thumbnail previews with modal playback
- UI sample image previews (OKA, E-label, Employee Advocacy)
- Google Drive & TikTok embed previews
- Mobile-first responsive layout

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Supabase Reviews (Optional)

Copy `.env.example` to `.env` and add your Supabase credentials to enable the Reviews tab:

```
VITE_SUPABASE_URL=your_url
VITE_SUPABASE_ANON_KEY=your_key
```

## Build

```bash
npm run build
npm run preview
```
